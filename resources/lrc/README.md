# Local LRC Files

Place your `.lrc` files in this folder (or configure a custom folder in Settings).

## Naming Convention

Files should be named using one of these patterns:

```
Artist - Title.lrc
Title - Artist.lrc
Title.lrc
```

For example:
- `Radiohead - Creep.lrc`
- `Imagine Dragons - Believer.lrc`

## LRC Format

Standard LRC format with timestamps:

```
[mm:ss.xx] Lyric text here
[00:19.16] When you were here before
[00:24.09] Couldn't look you in the eye
```

Both `[mm:ss.xx]` (centiseconds) and `[mm:ss.xxx]` (milliseconds) formats are supported.
